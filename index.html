cube.js

cube.js is a light script which allows you to add fancy rotating CSS3 cube to your page!

Basic Instructions

  1. Add either cube.js or cube.min.js script to your page.

    Example: <script src="cube.min.js"></script>

  2. Add styles responsible for the cube to you stylesheet(s).

    cube.css or cube.scss if you use Sass and Compass.
