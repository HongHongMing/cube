cube.js

cube.js is a light script which allows you to add fancy rotating CSS 3 cube to your page!

Basic Instructions

  1. Add either cube.js or cube.min.js script to your page.

    Example: <script src="cube.min.js"></script>

  2. Add styles responsible for the cube to you stylesheet(s).

    cube.css or cube.scss if you use Sass and Compass.

  3. Add cube's markup to your HTML:

    <div id="robot-cube" class="robot-cube">
	<span class="side-1"></span>
	<span class="side-2"></span>
	<span class="side-3"></span>
	<span class="side-4"></span>
	<span class="side-5"></span>
	<span class="side-6"></span>
</div>

  4. Add some styles to customise your cube:

    .robot-cube > span {
	background-image: url(images/cube/cube-texture.png);
}

  5. Inititate the script:

    var myCube = new Cube('#robot-cube');
	myCube.rotate({
		speedX: 10000,
		speedY: 10000,
		speedZ: 10000,
		startSide: 'side-1'
});

  6. And you are ready to go!
